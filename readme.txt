Dataset Link: https://www.kaggle.com/dmitrybabko/speech-emotion-recognition-en

DataPreprocessingAndAugmentation.ipynb: contains all the data preprocessing from the raw audios to the features extraction and also includes the augmentation as well as the mel-spectrogram images extraction.

Traditional_SVM_Approach.ipynb: contains the training of all the previously extracted features, comparing with/without augmentation using different models (but mainly, as presented in the report, the SVM)

DeepLearnigApproach.ipynb: constain the training of all the previously extracted mel-spectrogram images, using different DL models.
